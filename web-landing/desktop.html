Shield VPN Desktop (Windows Beta)

This desktop beta uses the official OpenVPN client to establish a system-wide VPN tunnel. When connected, all apps including Chrome, Edge, and other browsers will use the VPN — so it “works on Chrome desktop” by tunneling the system network stack.

Requirements

  1. Windows 10/11 (Admin user)
  2. OpenVPN community client installed
  3. At least one Shield VPN server config (.ovpn)

Quick Start

  1. Install OpenVPN (above).
  2. Download a server config from the Android app or request it via vpnapp@technosofts.net.
  3. Right-click the OpenVPN tray icon → Import file → choose the .ovpn.
  4. Click the tray icon → Connect. Enter username/password if prompted (provided with the config).
  5. Once connected, open Chrome — your traffic is now routed via the VPN.

Automate (optional)

You can automate connect/disconnect with OpenVPN's command line:

openvpn-gui --command connect <PROFILE_NAME>
openvpn-gui --command disconnect <PROFILE_NAME>

Replace <PROFILE_NAME> with the name shown in OpenVPN GUI.

Notes

  • This approach is the most reliable way to support Chrome/desktop today.
  • A native Flutter Desktop client is planned to provide a full UI and one-click connect experience.
  • If you need WireGuard support, we can provide .conf files and a similar flow using the WireGuard desktop app.
